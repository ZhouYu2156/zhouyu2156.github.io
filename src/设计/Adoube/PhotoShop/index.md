## 常用快捷键

| 快捷键                | 功能             | 转换功能 |
| :-------------------- | ---------------- | -------- |
| Ctrl + S              | 保存             |          |
| Ctrl + Z              | 撤销             |          |
| Ctrl + N              | 新建             |          |
| ESC                   | 取消             |          |
| Alt + DELETE          | 填充前景色       |          |
| Ctrl + Delete         | 填充背景色       |          |
| Ctrl + I              | 反向             |          |
| Shift + 框选          | 选区相加         |          |
| Alt + 框选            | 选区相减         |          |
| Alt + Shift + 框选    | 选区交叉         |          |
| Ctrl + 点击图层缩略图 | 载入选区         |          |
| Shift + X             | 前景和后景色切换 |          |
| Shift + 工具快捷键    | 切换工具类型     |          |
